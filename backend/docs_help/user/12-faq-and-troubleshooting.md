# FAQ and Troubleshooting

Common questions and solutions for Thesis users.

## Frequently Asked Questions

### General Questions

**Q: What can Thesis help me create?**

A: Thesis specializes in L&D deliverables including:
- Training curricula and program outlines
- Learning objectives and competency frameworks
- Assessments (knowledge checks, skill evaluations, scenarios)
- Facilitator guides and trainer notes
- Training visuals and images
- ROI measurement plans
- Needs analysis frameworks
- Presentation content

**Q: How is Thesis different from ChatGPT?**

A: Thesis is purpose-built for L&D professionals:
- Deep knowledge of instructional design, learning science, and performance consulting
- Built-in DDLD framework for ROI-driven design
- Specialized operating modes (Coach, Developer, Analyst, Advisor)
- Document upload and RAG for organization-specific context
- L&D-focused image generation
- Integration with your existing training materials

**Q: Is my data secure?**

A: Yes. Your data is:
- Encrypted at rest and in transit
- Private to your account (user documents)
- Not used to train AI models
- Stored securely with enterprise-grade protection

**Q: Can I use Thesis's outputs with clients?**

A: Yes. Content generated by Thesis can be:
- Used in client deliverables
- Modified and adapted as needed
- Branded with your organization's identity
- Included in training materials

We recommend reviewing and refining outputs before client delivery.

### Document Questions

**Q: What file types can I upload?**

A: Supported formats:
- PDF (.pdf) - up to 50 MB
- Word (.docx, .doc) - up to 50 MB
- PowerPoint (.pptx, .ppt) - up to 50 MB
- Text (.txt) - up to 10 MB

**Q: Why is my document still processing?**

A: Processing typically takes 30-60 seconds. If it's been longer:
1. Check if the file is corrupted (open it locally)
2. Verify it's under the size limit
3. Ensure it's a supported format
4. Try re-uploading the file

See [Uploading Documents](03-uploading-documents.md) for detailed troubleshooting.

**Q: How do I update an uploaded document?**

A: For direct uploads:
1. Delete the existing version
2. Upload the new version

For Google Drive/Notion:
1. Update the source file
2. Wait for scheduled sync, or click "Sync Now"

**Q: What's the difference between core documents and my documents?**

A: Core documents are organization-wide materials managed by administrators (visible to all users). Your documents are personal uploads visible only to you.

### Chat Questions

**Q: How do I find past conversations?**

A: Use the search field in the conversation sidebar. Search by:
- Keywords or topics discussed
- Client or project names
- Deliverable types ("facilitator guide", "assessment")

**Q: Can I export my conversations?**

A: Yes. Open a conversation and click the export icon. Available formats:
- Markdown
- Text
- PDF

For bulk export, use the conversation management features to select multiple conversations.

**Q: Why did Thesis's response seem off-topic?**

A: This can happen if:
- The conversation context drifted over many messages
- Your request was ambiguous
- Document context introduced unrelated content

Solutions:
- Start a new conversation for a fresh context
- Be more specific in your request
- Reference specific documents explicitly

**Q: How long can conversations be?**

A: Conversations can be quite long, but for best results:
- Start new conversations for distinct projects or topics
- Periodically summarize progress in long conversations
- Break large projects into multiple focused conversations

### Image Questions

**Q: What can I generate images of?**

A: Training visuals including:
- Workplace scenarios (office, retail, healthcare, manufacturing)
- Professional interactions (meetings, coaching, presentations)
- Concept visualizations (teamwork, leadership, growth)
- Diverse representation across age, ethnicity, and ability

**Q: What can't I generate?**

A: Limitations include:
- Readable text within images
- Real, named individuals
- Copyrighted logos or brands
- Photo-realistic specific locations
- Complex technical diagrams

**Q: How do I get better images?**

A: Be specific about:
- Who is in the image (roles, diversity, appearance)
- What they're doing (action, interaction)
- Where it's set (environment, lighting, mood)
- Style (professional, casual, energetic, calm)

See [Image Generation Guide](image-generation.md) for detailed tips.

### Project Questions

**Q: Should I use projects?**

A: Projects help if you:
- Work with multiple clients
- Have ongoing initiatives
- Want to organize conversation history
- Need to filter conversations by topic

If you primarily work on one thing at a time, projects are optional but still useful for organization.

**Q: Can I move conversations between projects?**

A: Yes. Open the conversation, click the project indicator, and select a different project.

**Q: What happens if I delete a project?**

A: The project is removed but conversations remain (they become unassigned). This cannot be undone.

### Account Questions

**Q: How do I change my password?**

A: Go to Profile Settings → Password section. Enter your current password, then your new password twice. See [Account Setup](01-account-setup.md).

**Q: Can I change my email address?**

A: Email changes require administrator assistance. Contact your admin.

**Q: What's View Mode?**

A: Admins can toggle between Admin View and User View to see the platform from a regular user's perspective. This doesn't change actual permissions.

## Troubleshooting

### Connection Issues

**Symptom:** Messages not sending, pages not loading.

**Solutions:**
1. Check your internet connection
2. Refresh the page
3. Clear browser cache
4. Try a different browser
5. Check if your organization has firewall restrictions

### Performance Issues

**Symptom:** Slow responses, long loading times.

**Solutions:**
1. Ensure stable internet connection
2. Close unnecessary browser tabs
3. Try a different browser
4. Clear browser cache and cookies
5. Reduce the complexity of your request

### Document Issues

**Symptom:** Document shows "Failed" status.

**Solutions:**
1. Open the file locally to verify it's not corrupted
2. Check the file is under size limits
3. Ensure it's a supported format
4. Try saving as a different format (e.g., Word to PDF)
5. Re-upload the file

**Symptom:** Documents not appearing in search results.

**Solutions:**
1. Verify document status is "Completed"
2. Check that document content is relevant to your query
3. Use more specific search terms
4. Ensure document text is selectable (not scanned image)

### Integration Issues

**Symptom:** Google Drive not syncing.

**Solutions:**
1. Check sync cadence settings (may be set to Manual)
2. Verify permissions weren't revoked
3. Disconnect and reconnect the integration
4. Ensure files in Drive are supported formats

**Symptom:** Notion pages not updating.

**Solutions:**
1. Verify the page is shared with Thesis during authorization
2. Check sync cadence settings
3. Trigger manual sync
4. Ensure page content is in supported format (text, not embeds)

### Image Generation Issues

**Symptom:** Image generation fails.

**Solutions:**
1. Simplify your prompt
2. Remove potentially problematic content (real people, brands)
3. Try generating again
4. Check internet connection

**Symptom:** Image doesn't match description.

**Solutions:**
1. Add more specific details
2. Describe what's wrong with current image
3. Request different perspective or composition
4. Generate multiple variations

### Login Issues

**Symptom:** Can't sign in.

**Solutions:**
1. Verify email and password
2. Use "Forgot Password" to reset
3. Clear browser cookies for the site
4. Try incognito/private browsing mode
5. Contact administrator if issues persist
