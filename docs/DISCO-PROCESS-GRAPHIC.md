# DISCo Process Flow (Graphic Reference)

## Why DISCo?

**The Problem:** Product discovery generates mountains of transcripts, notes, and research that never become actionable. Weeks of workshops produce 50-page decks that executives don't read. Decisions stall. Good ideas die in the gap between "we talked about it" and "here's what we're doing."

**The Solution:** DISCo transforms discovery chaos into decision clarity in days, not months.

| Without DISCo | With DISCo |
|---------------|------------|
| 50-page discovery docs nobody reads | 900-word decision doc, decision in first sentence |
| Weeks to synthesize findings | Hours from transcript to recommendation |
| Vague "we should explore" conclusions | Clear GO/NO-GO with owner and deadline |
| No accountability (who does what?) | Real names, real actions, real dates |
| Executives ask "so what?" | 30-second clarity on what to do next |
| Discovery that never ends | Structured loop with clear "ready" criteria |

**For Our Team:**
- **Chris** gets measurable outcomes with STAR-format accountability
- **Tyler** gets structured frameworks he can learn and reuse
- **Mikki** gets systems thinking with intentional problem definition

## Six Agents, One Flow

| # | Agent | Role | Time | Output |
|---|-------|------|------|--------|
| 1 | **Triage** | Intake gate | 5 min | GO / NO-GO / INVESTIGATE |
| 2 | **Discovery Planner** | Design sessions | 10 min | Session agendas + questions |
| 3 | **[HUMANS]** | Run sessions | Hours | Transcripts + notes |
| 4 | **Coverage Tracker** | Check readiness | 5 min | READY / GAPS REMAIN |
| 5 | **Insight Extractor** | Extract meaning | 10 min | Patterns + evidence |
| 6 | **Synthesizer** | Final decision | 15 min | 900-word decision doc |

## The Discovery Loop

```
         ┌────────────────────────────────────┐
         │                                    │
         ▼                                    │
   DISCOVERY  ───▶  HUMAN   ───▶  COVERAGE   ─┤
   PLANNER         SESSION        TRACKER     │
                                    │         │
                                    │   GAPS REMAIN
                              READY │         │
                                    ▼         │
                             INSIGHT    ◀─────┘
                             EXTRACTOR
```

**Key Insight:** Run Coverage Tracker after EVERY session. Iterate until "READY FOR SYNTHESIS."

## What Each Agent Produces

| Agent | Input | Output |
|-------|-------|--------|
| Triage | Problem statement | GO/NO-GO + tier routing |
| Discovery Planner | GO decision | Session plan with questions |
| Coverage Tracker | Transcripts | Gap analysis + readiness status |
| Insight Extractor | All transcripts | Patterns, contradictions, key quotes |
| Synthesizer | Insights | Decision doc with leverage point, metrics, actions |
