# Theme Customization

This guide covers all aspects of customizing Thesis's appearance, from color palettes to typography and branding.

## Accessing Theme Settings

Navigate to **Dashboard** → **Theme** to open the **Theme Settings** page.

## Action Buttons

Two primary buttons appear at the top:
- **Reset to Defaults** - Restore all settings to original values
- **Save Changes** - Apply your modifications (shows "Saving..." while processing)

When you have unsaved modifications, a warning appears: "You have unsaved changes"

## Color Palette Presets

The **Color Palette Presets** section provides 15 pre-designed WCAG AA accessible color schemes for quick setup.

### Available Presets

Click any preset to apply its complete color scheme:

**Dark Themes:**
- **Indigo Professional** - Clean, modern look with indigo accents
- **Ocean Blue** - Calming blue tones for a trustworthy feel
- **Forest Green** - Natural, eco-friendly appearance
- **Royal Purple** - Elegant and creative styling
- **Warm Coral** - Friendly and inviting warm tones
- **Slate Minimal** - Clean, neutral business style
- **Rose Gold** - Sophisticated with warm pink accents
- **Teal Modern** - Fresh and contemporary teal theme
- **Amber Warmth** - Golden amber for a welcoming feel

**Light Themes:**
- **Light Professional** - Light theme for daytime use
- **Light Teal** - Fresh teal on clean white
- **Light Purple** - Elegant purple on soft lavender
- **Light Green** - Natural green on soft mint
- **Light Warm** - Cozy amber on warm cream
- **Light Rose** - Romantic pink on blush

### Applying a Preset

1. Browse the preset options
2. Click on a preset name
3. All colors update to match the preset
4. Click **Save Changes** to apply

A success message confirms: 'Applied "[Preset Name]" palette'

## Custom Colors

The **Colors** section allows fine-grained control over every color in the interface.

### Brand Colors

Configure your primary brand colors:
- **Primary** - Main accent color (buttons, links, highlights)
- **Primary Hover** - Color when hovering over primary elements
- **Secondary** - Alternative accent color

### Background Colors

Set page and component backgrounds:
- **Page** - Main page background
- **Card** - Panel and card backgrounds
- **Hover** - Background for hover states

### Panel Text Colors

Configure text appearance within panels:
- **Headings** - Titles, headers
- **Body** - Content text
- **Captions** - Subtitles, hints

### Button Text Colors

Set text colors on buttons:
- **Primary Btn** - Text on primary colored buttons
- **Secondary Btn** - Text on secondary/outline buttons

### Status Colors

Configure feedback and status indicators:
- **Success** - Positive actions and confirmations
- **Warning** - Cautions and alerts
- **Error** - Errors and problems

### Using the Color Picker

Each color setting has two input methods:
1. **Color swatch** - Click the colored square to open a visual picker
2. **Hex input** - Type a hex color code directly (e.g., #4f46e5)

### Colors Preview

The preview section shows how your colors appear on:
- **Primary Button** - Sample button with primary color
- **Success** badge
- **Warning** badge
- **Error** badge

## Top Nav Bar Customization

The **Top Nav Bar** section controls the header appearance.

### Logo

Upload your organization's logo:
- Supported formats: **PNG**, **JPEG**, **GIF**, **SVG**, **WebP**
- Maximum file size: **2MB**

**To upload:**
1. Click **Upload Logo**
2. Select your image file
3. Logo appears in the header

**To remove:**
1. Click **Remove** next to the current logo
2. Confirm removal

### Header Colors

Configure header appearance:
- **Background** - Header bar background color
- **Title Color** - Color of the title/brand text
- **Nav Links** - Color of navigation link text
- **Nav Active** - Color of the currently active navigation link

### Header Sizing

Adjust header dimensions:
- **Title Font Size** - Choose from 12px to 32px options
- **Bar Height** - Choose from 48px to 80px options:
  - **48px** - Compact
  - **56px** - Small
  - **64px** - Medium (default)
  - **72px** - Large
  - **80px** - Extra large

### Header Preview

A live preview shows your header settings:
- Displays logo or title text
- Shows active and inactive navigation links
- Updates in real-time as you adjust settings

## Page Title Settings

The **Page Title** section controls how page headings appear.

### Font Size

Select page title size from:
- **20px**
- **24px**
- **28px**
- **32px**
- **36px**
- **40px**
- **48px**

### Page Title Preview

A preview panel shows how page titles will appear with your selected size and the heading font settings from Typography.

## Typography Settings

The **Typography** section controls fonts and text styling.

### Heading Font

Choose the font for titles and headers:
- **Heading Font** dropdown - Select from available fonts
- **Weight** dropdown - Choose from:
  - **Regular**
  - **Medium**
  - **Semi Bold**
  - **Bold**

### Body Font

Choose the font for content text:
- **Body Font** dropdown - Select from available fonts
- **Weight** dropdown - Same weight options as headings

### Available Fonts

Both heading and body fonts can use:
- **Inter**
- **Roboto**
- **Open Sans**
- **Lato**
- **Poppins**
- **Montserrat**
- **Source Sans Pro**
- **Nunito**
- **Raleway**
- **Ubuntu**
- **System Default**

### Base Font Size

Set the default text size:
- **14px**
- **15px**
- **16px** (recommended default)
- **17px**
- **18px**

### Typography Preview

A preview panel shows:
- **Heading Text** in your selected heading font
- Body text in your selected body font
- Both update in real-time as you change settings

## Borders and Panels

The **Borders & Panels** section controls element styling.

### Border Colors

Configure border appearances:
- **Default** - Standard border color
- **Focus** - Border color when elements are focused

### Border Radius

Control corner rounding at three levels:
- **Small** - For small elements like buttons
- **Medium** - For cards and panels
- **Large** - For major containers

Each can be set to:
- **None**
- **Extra Small**
- **Small**
- **Medium Small**
- **Medium**
- **Large**
- **Extra Large**

### Panel Styling

Configure panel edge effects:
- **Border Width** - None, 1px, 2px, or 3px
- **Shadow Size** - None, Small, Medium, Large, or Extra Large

### Panel Colors

Fine-tune panel appearance:
- **Panel Border Color** - Color of panel borders
- **Panel Shadow Color** - Color of panel shadows

### Borders & Panels Preview

A preview panel demonstrates:
- Your border radius settings at different sizes
- Panel border and shadow effects
- How text appears within styled panels

## Saving Your Theme

### Applying Changes

After making any modifications:
1. Review the preview sections
2. Click **Save Changes**
3. Wait for the confirmation message
4. The new theme applies across the entire platform

### Unsaved Changes Warning

If you try to leave with unsaved changes, you'll see: "You have unsaved changes"

### Resetting to Defaults

To restore original settings:
1. Click **Reset to Defaults**
2. Confirm the reset
3. All settings return to their original values
4. Success message: "Theme reset to defaults"

## Theme Best Practices

### Accessibility

All preset palettes are WCAG AA compliant. When creating custom colors:
- Ensure sufficient contrast between text and backgrounds
- Test with different lighting conditions
- Consider users with color vision differences

### Branding Consistency

1. Use your organization's official brand colors
2. Match fonts to your brand guidelines
3. Upload a high-quality logo

### Testing Changes

Before saving:
1. Review all preview sections
2. Check both dark and light areas
3. Verify text readability
4. Test button and status colors

### Logo Guidelines

For best results:
- Use a transparent PNG or SVG
- Keep file size under 2MB
- Ensure logo is legible at small sizes
- Test logo on both light and dark backgrounds

## Common Theme Tasks

### Applying Your Brand Colors

1. **Dashboard** → **Theme**
2. Scroll to **Colors** → **Brand**
3. Enter your brand's **Primary** color hex code
4. Adjust **Primary Hover** to a slightly darker shade
5. Set **Secondary** to a complementary color
6. Click **Save Changes**

### Uploading Your Company Logo

1. **Dashboard** → **Theme**
2. Scroll to **Top Nav Bar** → **Logo**
3. Click **Upload Logo**
4. Select your logo file
5. Verify it appears correctly in the preview
6. Click **Save Changes**

### Switching to a Light Theme

1. **Dashboard** → **Theme**
2. Find **Color Palette Presets**
3. Click **Light Professional** or another light preset
4. Click **Save Changes**

### Adjusting Font Sizes for Readability

1. **Dashboard** → **Theme**
2. Scroll to **Typography**
3. Increase **Base Font Size** (e.g., to 17px or 18px)
4. Review the preview
5. Click **Save Changes**
