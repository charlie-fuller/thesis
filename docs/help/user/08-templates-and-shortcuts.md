# Templates and Shortcuts

Speed up your work with pre-built templates and command shortcuts.

## Command Shortcuts

Type these commands at the start of your message for quick actions:

### `/assess`

Generate assessments quickly.

**Usage:**
```
/assess customer service skills
/assess leadership competencies for new managers
/assess technical knowledge of safety protocols
```

**What you get:**
- Knowledge check questions (multiple choice, true/false)
- Scenario-based questions
- Performance-based assessment criteria
- Answer keys with explanations

### `/roi`

Create ROI measurement plans.

**Usage:**
```
/roi sales training program
/roi leadership development initiative
/roi compliance training refresh
```

**What you get:**
- Key performance indicators to track
- Data collection methods
- Before/after measurement approach
- ROI calculation framework
- Reporting recommendations

### `/outline`

Generate program outlines.

**Usage:**
```
/outline 2-day sales training
/outline new hire onboarding program
/outline customer service workshop
```

**What you get:**
- Module structure with timing
- Learning objectives per section
- Activity suggestions
- Assessment checkpoints
- Materials list

### `/image`

Quick image generation.

**Usage:**
```
/image diverse team collaborating
/image manager coaching employee
/image customer service scenario
```

**What you get:**
- Generated training visual (3-5 seconds)
- Default to landscape format

For more control over images, use natural language requests with aspect ratio and style specifications.

## Using Commands Effectively

### Combine with Context

Add details after the command:

```
/assess customer service skills for healthcare front desk staff,
focusing on patient communication and HIPAA awareness
```

```
/outline 4-hour workshop on giving feedback,
for first-time managers in a tech company
```

### Follow Up Naturally

After using a command, continue the conversation normally:

```
User: /outline leadership training
Thesis: [generates outline]
User: Add more content on emotional intelligence in module 2
Thesis: [updates outline]
```

## Templates Library

Access pre-built templates for common L&D tasks.

### Accessing Templates

1. Click **Templates** in the navigation or sidebar
2. Browse available templates by category
3. Click a template to preview
4. Click **Use Template** to start a conversation with that template

### Template Categories

| Category | Examples |
|----------|----------|
| **Needs Analysis** | Stakeholder interview guide, Performance gap analysis |
| **Curriculum Design** | Course outline template, Module planning framework |
| **Assessment** | Knowledge check templates, Skill assessment rubric |
| **Facilitation** | Facilitator guide template, Activity design framework |
| **Evaluation** | Level 1 survey template, ROI measurement plan |

### How Templates Work

When you use a template:

1. A new conversation starts with the template context
2. Thesis understands what you're trying to create
3. You provide your specific details
4. Thesis generates customized output

**Example Template Flow:**

```
Template: "Needs Analysis Interview Guide"

Thesis: "I'll help you create a stakeholder interview guide.
What's the business challenge or performance issue you're investigating?"

User: "Sales team is missing quota consistently"

Thesis: [Generates customized interview questions for sales performance]
```

## Quick Prompts

Beyond commands, these prompt patterns get fast results:

### For Assessments

```
"Create 10 multiple choice questions about [topic] for [audience]"
"Design a scenario-based assessment for [skill] with scoring rubric"
"Build a pre/post knowledge check for [program]"
```

### For Content Development

```
"Write learning objectives for [topic] using Bloom's taxonomy"
"Create an activity that teaches [concept] through practice"
"Design a role-play scenario for [skill]"
```

### For Facilitation

```
"Create a facilitator guide for [activity]"
"Design an icebreaker for [audience]"
"Write discussion questions for [topic]"
```

### For Visuals

```
"Generate a process diagram showing [workflow]"
"Create an image for teaching [concept]"
"Design a scenario background for [situation]"
```

## Building Your Own Quick Patterns

Develop personal shortcuts by noting what works:

### Create a Personal Library

Keep track of effective prompts:

```
My Assessment Prompt:
"Create a [count] question assessment for [audience] on [topic].
Include [question types]. Align with [objectives]."

My Outline Prompt:
"Design a [duration] training for [audience] on [topic].
Include objectives, activities, and assessment for each module."
```

### Use Consistent Structure

Develop templates for your frequent requests:

```
[Deliverable type] for [audience] on [topic]
Context: [client/industry specifics]
Constraints: [time, format, requirements]
Include: [specific elements]
```

### Reference Documents

Combine templates with your uploaded materials:

```
"Using our competency framework document, /assess leadership skills
for mid-level managers ready for promotion"
```

## Efficiency Tips

### Batch Similar Requests

When you need multiple similar outputs:

```
"Create learning objectives for all 5 modules:
1. Introduction to Feedback
2. Giving Constructive Feedback
3. Receiving Feedback
4. Feedback in Difficult Situations
5. Building a Feedback Culture"
```

### Chain Commands

Build comprehensive deliverables step by step:

```
1. /outline sales training
2. "Expand module 3 with detailed activities"
3. /assess for module 3 content
4. "Create facilitator notes for the main activity"
```

### Use Templates as Starting Points

Don't feel locked into template structure:

```
User: [Uses assessment template]
Thesis: [Generates assessment]
User: "Change to all scenario-based questions instead of multiple choice"
Thesis: [Revises format]
```

## Common Workflow Patterns

### Complete Course Development

1. `/outline [course topic]` - Get structure
2. "Expand learning objectives for each module"
3. "Design activities for module [n]"
4. `/assess [module topic]` - Create assessments
5. `/image [scene]` - Generate visuals
6. "Create facilitator guide"

### Quick Deliverable

1. `/assess [topic]` or `/outline [topic]`
2. "Customize for [audience specifics]"
3. Export or copy the result

### ROI-Focused Design

1. `/roi [initiative]` - Start with measurement plan
2. "What training would move these metrics?"
3. `/outline` - Build program aligned to ROI
