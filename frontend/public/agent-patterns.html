Agent Interaction Patterns

Five ways to work with AI agents in Thesis

Interaction Modes
1
Single Agent Mode
Lock to one agent for a deep-dive on a specific topic. You select the agent explicitly from the agent selector, and all messages in that conversation go directly to that agent.
How it works
Select an agent from the dropdown in the chat sidebar. Every message you send stays with that agent until you switch.
Best for: Focused research, financial analysis, legal review
2
Auto-Routing
Let the system pick the best agent based on your question. The router analyzes your message and dispatches to the most relevant agent automatically.
How it works
Leave the agent selector on "Auto" and type your question. The routing system classifies your intent and selects the agent with the best match.
Best for: Quick questions when you don't know which agent to ask
3
@Mentions
Tag specific agents mid-conversation for targeted input. Type @AgentName to pull a specific agent into an existing conversation without switching context.
How it works
While chatting with any agent, type @ followed by an agent name. That agent responds to your current message, then control returns to the original agent.
Best for: Getting a second opinion or specialized input
4
Meeting Rooms
Set up multi-agent autonomous discussions. Create a room with a topic and select participant agents. Agents discuss autonomously in 50-100 word turns, building on each other's contributions.
How it works
Create a Meeting Room, set a discussion topic, and add 2-6 agents. Start the session and agents take turns contributing perspectives. You can interject at any point.
Best for: Brainstorming, exploring trade-offs, competing perspectives
5
Kraken (Task Automation)
AI evaluates tasks and recommends agent-assisted approaches. Kraken analyzes task context, description, and linked documents, then suggests which agent can help and how to approach the work.
How it works
Open any task card and click "Release the Kraken." Kraken reviews the task details, identifies the most relevant agent, and proposes an action plan you can accept, modify, or dismiss.
Best for: Breaking down tasks, getting agent recommendations for specific work items
Which mode should I use?
Match your intent to the right interaction mode. If you're unsure, start with Auto-Routing -- the system will find the right agent for you.
"I need a quick answer" → Auto-Routing
"I need deep analysis" → Single Agent
"I want a second opinion" → @Mention
"I want competing perspectives" → Meeting Room
"I have a task to complete" → Kraken
