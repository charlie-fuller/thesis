Quick Start Guide

Everything you need to know in one page

First 3 Things to Do

  1. 1 Upload your first documents to the Knowledge Base (drag-and-drop or sync your local vault)
  2. 2 Create your first DISCO initiative and link a document folder
  3. 3 Run the Discovery Guide to triage your initiative (GO / NO-GO / INVESTIGATE)

Tips

  • Auto-routing works well for quick questions -- you don't always need to pick an agent
  • Link document folders (not individual docs) to DISCO initiatives for best coverage
  • Run Coverage mode after every discovery session to track what's addressed
  • The golden path: KB docs -> Initiative -> Pipeline -> Project -> Tasks -> Done
  • Every task traces back to its source through the full traceability spine
